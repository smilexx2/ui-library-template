# Installation

## Prerequisites

Before installing the UI Library, make sure you have the following:

- **Node.js** 16.8 or later
- **React** 16.8 or later
- **React DOM** 16.8 or later

## Install the Library

Install the custom button component:

```bash
npm install @smilexx2/custom-button
```

For utility functions (optional):

```bash
npm install @smilexx2/ui-core
```

## Setup Tailwind CSS

This library uses Tailwind CSS for styling. Make sure you have Tailwind CSS installed and configured in your project.

```bash
npm install -D tailwindcss
npx tailwindcss init
```

Add the library's styles to your Tailwind config:

```js
// tailwind.config.js
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/@smilexx2/custom-button/dist/**/*.js",
    "./node_modules/@smilexx2/ui-core/dist/**/*.js" // if using utilities
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

## Usage

Once installed, you can import and use components in your React application:

```tsx
import { CustomButton } from '@smilexx2/custom-button'
// Optional: import utilities
import { cn } from '@smilexx2/ui-core'

function MyApp() {
  return (
    <div className="p-4 space-y-4">
      <CustomButton variant="default">Click me</CustomButton>
      <CustomButton variant="outline">Enhanced button</CustomButton>
      <CustomButton variant="secondary">Secondary button</CustomButton>
    </div>
  )
}
```

## TypeScript Support

The library is written in TypeScript and includes full type definitions. No additional setup is required for TypeScript projects.
