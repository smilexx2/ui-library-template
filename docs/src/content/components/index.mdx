- Screen reader compatibility

### Performance
- Optimized bundle sizes
- Tree-shakeable exports

- Minimal runtime overhead

### Developer Experience
- TypeScript definitions
- Comprehensive prop documentation
- Interactive examples
- Copy-paste friendly code

## Getting Started

To start using our components:

1. Install the packages you need
2. Import the components and styles
3. Use them in your React application

```tsx
import { CustomButton } from '@smilexx2/custom-button'
import '@smilexx2/custom-button/styles.css'

function App() {
  return (
    <CustomButton variant="default">
      Get Started
    </CustomButton>
  )
}
```
