# Custom Button

An enhanced button component with additional features and animations that extends the standard button functionality.

## Installation

```bash
npm install @smilexx2/custom-button
```

## Usage

The CustomButton component provides enhanced features with smooth animations and additional styling options.

```tsx
import { CustomButton } from '@smilexx2/custom-button';

<CustomButton variant="default">Click me</CustomButton>
<CustomButton variant="secondary">Secondary Action</CustomButton>
<CustomButton variant="outline">Outline Style</CustomButton>
```

## Sizes

CustomButton supports multiple sizes for different use cases.

```tsx
<CustomButton size="sm">Small Button</CustomButton>
<CustomButton size="default">Default Button</CustomButton>
<CustomButton size="lg">Large Button</CustomButton>
```

## API Reference

### CustomButton Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "outline" \| "destructive" \| "ghost" \| "link" \| "gradient" \| "glow" \| "neon"` | `"default"` | The visual style variant |
| `size` | `"xs" \| "sm" \| "default" \| "lg" \| "xl" \| "icon"` | `"default"` | The size of the button |
| `animation` | `"none" \| "pulse" \| "bounce" \| "wobble" \| "ripple"` | `"none"` | Animation type |
| `loading` | `boolean` | `false` | Show loading spinner and disable button |
| `leftIcon` | `ReactNode` | - | Icon to display on the left |
| `rightIcon` | `ReactNode` | - | Icon to display on the right |
| `ripple` | `boolean` | `false` | Enable ripple effect on click |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Button content |
| `disabled` | `boolean` | `false` | Disable the button |
| `onClick` | `() => void` | - | Click handler |

## Features

- **Enhanced Animations**: Smooth hover and active state transitions
- **TypeScript Support**: Full type safety with TypeScript definitions
- **Accessible**: ARIA compliant with keyboard navigation support
- **Customizable**: Supports className prop for additional styling
- **Theme Compatible**: Works with your existing theme system
- **Loading States**: Built-in loading spinner with automatic disabling
- **Ripple Effects**: Optional click ripple animation
- **Icon Support**: Easy left and right icon placement

## Styling

The CustomButton component automatically includes all necessary styles when imported. It uses CSS variables for theming and will automatically adapt to your application's color scheme.

No additional CSS imports are required - the component is completely self-contained.
