# UI Library Documentation

A personal component library extending shadcn/ui with React and TypeScript.

## Getting Started

Install the library:

```bash
npm install @smilexx2/ui-library
```

## Components

This library includes the following components:

- **Avatar** - User profile pictures with fallback support
- **DataTable** - Table component for displaying data
- **FileUpload** - File upload interface
- **Spinner** - Loading indicators
- **StatsCard** - Cards for displaying statistics
- **UserProfile** - User profile display components

## Usage

Import components from the library:

```tsx
import { Avatar, DataTable, Spinner } from '@smilexx2/ui-library';
```
