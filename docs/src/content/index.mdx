## Quick Start

### Installation

Choose the packages you need:

```bash
# Core utilities
npm install @smilexx2/ui-core

# shadcn/ui style components
npm install @smilexx2/ui

# Enhanced components
npm install @smilexx2/custom-button
```

### Basic Usage

```tsx
import { cn } from '@smilexx2/ui-core'
import { Button } from '@smilexx2/ui'
import { CustomButton } from '@smilexx2/custom-button'

// Import styles
import '@smilexx2/ui/styles.css'
import '@smilexx2/custom-button/styles.css'

function App() {
  return (
    <div className={cn("space-y-4 p-8")}>
      <Button variant="outline">Standard Button</Button>
      <CustomButton variant="default">Enhanced Button</CustomButton>
    </div>
  )
}
```

## Why @smilexx2 UI?
