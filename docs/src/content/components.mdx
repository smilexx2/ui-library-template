# Components

## CustomButton

An enhanced button component with smooth animations and additional styling options.

```tsx
import { CustomButton } from '@smilexx2/custom-button';

<CustomButton variant="default">Click me</CustomButton>
<CustomButton variant="outline">Outline button</CustomButton>
```

## Avatar

User profile pictures with fallback support.

```tsx
import { Avatar } from '@smilexx2/ui-library';

<Avatar src="/profile.jpg" alt="User" fallback="JD" />
```

## DataTable

Table component for displaying data.

```tsx
import { DataTable } from '@smilexx2/ui-library';

<DataTable data={tableData} columns={columns} />
```

## FileUpload

File upload interface component.

```tsx
import { FileUpload } from '@smilexx2/ui-library';

<FileUpload onUpload={handleUpload} />
```

## Spinner

Loading indicators for async operations.

```tsx
import { Spinner } from '@smilexx2/ui-library';

<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />
```

## StatsCard

Cards for displaying statistics and metrics.

```tsx
import { StatsCard } from '@smilexx2/ui-library';

<StatsCard title="Total Users" value="1,234" />
```

## UserProfile

User profile display components.

```tsx
import { UserProfile } from '@smilexx2/ui-library';

<UserProfile name="John Doe" email="john@example.com" />
```
