# shadcn/ui Extensions

A collection of extension components that build on top of shadcn/ui.

## Features

- **Extension-based**: Build on top of existing shadcn/ui components
- **Peer dependencies**: Use your existing shadcn/ui installation
- **Interactive demos**: Live code editing with React Live
- **TypeScript**: Full TypeScript support

## Installation

```bash
npm install @your-username/ui-library
```

## Quick Start

All components extend shadcn/ui and inherit its theming system:

```tsx
import { StatsCard } from "@your-username/ui-library"

function App() {
  return (
    <StatsCard
      title="Total Revenue"
      value="$12,345"
      trend={{ value: 12, label: "vs last month" }}
    />
  )
}
```
